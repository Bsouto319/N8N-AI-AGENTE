# N8N-AI-AGENTE
Agente de IA para atendimento automático no WhatsApp usando n8n, Google Gemini e Redis. Inclui webhook, memória de conversação e respostas inteligentes.   Funcionalidades: • Integração WhatsApp via WAHA • IA Google Gemini para respostas • Memória Redis para contexto • Webhook para mensagens • Respostas automáticas
